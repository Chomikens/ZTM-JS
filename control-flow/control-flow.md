# Conditionals 

## `If statement `

### Syntax 

```js 
if (statement) {
    // do something
}  else {
    // do something else
}
```

### Way to impove: 

```js
 if(statement) // do something
 if(anotherStatement) // do something else 
 ```

#### Practical example: 

```js 
const birthMonth = 12 

if (birthMonth === 12) {
    console.log('You lucky bastard - you wil get two presents')
} else {
    console.log(' I like patato ~ Johny Bravo')
}
```
